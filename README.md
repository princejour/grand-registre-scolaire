# الدفتر الكبير للغيابات

تطبيق ويب عربي لإعداد كتاب مستقل لكل مستوى: السابعة، الثامنة، والتاسعة.

## الوظائف

- استيراد عدة قائمات بصيغة CSV أو Excel.
- قراءة كل ورقة Excel كقائمة قسم مستقلة.
- اكتشاف المستوى ورقم القسم آليًا من اسم الملف أو ورقة Excel أو محتوى القائمة.
- إمكانية تصحيح المستوى واسم القسم يدويًا.
- تحديد السنة الدراسية وضبط العطل المدرسية والوطنية والدينية.
- تظليل أيام الأحد والعطل آليًا.
- صفحة مستقلة لكل شهر من أكتوبر إلى ماي.
- إصدار كتاب PDF مستقل لكل مستوى.
- طباعة مباشرة بتنسيق A3 أفقي وهوامش تجليد متناوبة.
- حفظ القائمات والإعدادات محليًا على الجهاز.

## أفضل تسمية للقائمات

```text
السابعة-1.xlsx
الثامنة-4.csv
التاسعة-2.xlsx
```

يدعم التطبيق أيضًا تسميات مثل `7A1.xlsx` و`8eme-4.xlsx` و`9eme-2.xlsx`.

## الطباعة

- المقاس: A3
- الاتجاه: أفقي Landscape
- الطباعة على الوجهين
- التقليب على الحافة القصيرة
- المقياس: 100%

## الرزنامة

يجب مراجعة العطل المدرسية والأعياد الدينية طبقًا للرزنامة الرسمية للسنة المختارة قبل إصدار النسخة النهائية.
